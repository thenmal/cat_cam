About μ-π

μ-π is a project created for the Rasberry Pi BakeOff at Hendrix College. Inspired by Pet Collective's Kitten Cam and Chrome Web Lab Experiment Universal-Orchestra, Cat Game combines interactivity with the joy of watching live cats!

We are using a Raspberry Pi to run this whole site! It not only runs the webcam, but also runs the webserver. When you click the 'Play with the cats' button, the Pi signals an Arduino to move a servo atached to a laser pointer.
